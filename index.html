How to run Install Node.js first. Run: npm install npm run dev # 2. Stack & design choices I used React with Vite because it made live updates easier while keeping the project simple and fast. One design choice was using a two-column layout on desktop so users can always see the results while editing inputs. Another decision was making the preset tip buttons visually distinct when active so users immediately know which percentage is selected. # 3. Responsive & accessibility On a 360px phone screen the layout stacks vertically so inputs and results are easier to read and tap. On a 1440px screen the app uses two columns to better use space. One accessibility feature I included was visible labels for every input field. One thing I did not fully handle was advanced screen reader announcements for validation errors because I focused more on the visual interaction. # 4. AI usage I used ChatGPT to help structure the React component and plan the validation logic. One thing I changed from the AI output was the layout styling. I simplified the spacing and reduced extra sections so the interface felt cleaner and easier to use. # 5. Honest gap One thing that still needs improvement is handling extremely large pasted numbers more gracefully. With another day I would improve number formatting and add better handling for unusual edge cases. ```
